URL Shortener Microservice

Documentation

  • Valid URLs can be sent (via POST) to to /api/shorturl/new and the server will respond with a JSON object with the shortened URL
  • The server will handle invalid URLs by responding with a JSON object containing an error message

Example of usage

POST 'https://www.google.com/' to /api/shorturl/new, which will provide a shortened url that will redirect to google

Example Output (JSON)

{
"url": "https://www.google.com/",
"short_url": "[this_project_url]/api/shorturl/1"
}
